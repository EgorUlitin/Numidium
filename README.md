# Numidium

Roadmap and tasks on [Trello board](https://trello.com/b/tOSS7V7z).

## Development
+ `git clone git@github.com:TheNovel/Numidium.git`
+ `cd Numidium/app`
+ `composer install -o`
+ `php bin/console doctrine:database:create`
+ `php bin/console server:start`
+ `yarn install`
+ `yarn watch`
